# SQL Complex Joins & Reporting

## Objetivo
Practicar consultas relacionales avanzadas utilizando múltiples tablas y JOINs para generar informes de negocio.

## Funcionalidades
- Claves foráneas e integridad referencial
- JOIN entre Clientes, Ventas y Articulos
- Filtros con IN, LIKE y condiciones compuestas
- Consultas de reporting

## Estructura
schema.sql  → definición de tablas
seed.sql    → datos de ejemplo
queries.sql → consultas analíticas

## Uso
mysql -u root -p < schema.sql
mysql -u root -p < seed.sql
mysql -u root -p < queries.sql
