# Inventory Database Schema (SQL)

## Objetivo
Diseñar una base de datos relacional para gestionar el inventario de productos mediante SQL.

## Funcionalidades implementadas
- Creación de tabla Productos
- Clave primaria
- Restricciones NOT NULL
- Tipos de datos adecuados (INT, VARCHAR, DECIMAL)
- Inserción de registros
- Posible ampliación con ALTER TABLE

## Tecnologías
- SQL (DDL + DML)
- MySQL / MariaDB compatible

## Archivos
- schema.sql → estructura de tablas
- seed.sql → datos de ejemplo

## Resultado
Base de datos simple y escalable para gestión de inventario.
