# Remote Access Management (RealVNC)

## Objetivo
Configurar acceso remoto seguro a un PC desde un dispositivo móvil para tareas de administración y soporte técnico.

## Tareas realizadas
- Instalación de VNC Server en el PC
- Configuración de autenticación y contraseña
- Activación de cifrado AES
- Instalación de VNC Viewer en móvil
- Conexión remota y control del equipo
- Cierre seguro de sesiones
- Restricción de accesos no autorizados

## Medidas de seguridad aplicadas
- Contraseña remota
- Cifrado AES
- Verificación de identidad del host
- Desconexión manual de sesiones activas

## Tecnologías
- RealVNC Server
- VNC Viewer (Android/iOS)

## Resultado
Acceso remoto funcional, seguro y multiplataforma para administración del sistema.
