# Descarga P2P segura y verificación de integridad

## Objetivo
Configurar un cliente BitTorrent de forma segura y verificar la integridad de archivos descargados mediante hash y análisis antivirus.

## Tareas realizadas
- Instalación del cliente uTorrent
- Configuración de carpetas de descarga
- Limitación de ancho de banda
- Cifrado de protocolo BitTorrent (Forced encryption)
- Excepción controlada en firewall
- Descarga mediante enlace Magnet
- Verificación de seguridad con:
  - Microsoft Defender
  - VirusTotal
  - Comprobación de hash SHA-256

## Herramientas usadas
- uTorrent
- Windows Defender
- VirusTotal
- Hash SHA-256

## Resultado
Archivo verificado como íntegro (hash coincidente) y sin detecciones de malware.

## Enfoque de seguridad
Se aplicaron buenas prácticas para minimizar riesgos al usar redes P2P:
- cifrado
- validación de integridad
- análisis antivirus
- control de consumo de red
